# Quick Deploy Instructions

## Fastest Method: Netlify (5 minutes)

1. Go to [netlify.com](https://netlify.com) and sign up
2. Drag this entire folder onto the Netlify dashboard
3. Done! You'll get a URL like `https://your-app.netlify.app`

## GitHub Pages Method

1. Create a GitHub repository
2. Upload all files from this `web` folder
3. Go to Settings → Pages → Select `main` branch
4. Your app will be at `https://YOUR_USERNAME.github.io/REPO_NAME`

## File Structure

Make sure these files are included:
- ✅ index.html
- ✅ styles.css
- ✅ js/ folder
- ✅ data/ folder

That's it! Your app is ready to share.
